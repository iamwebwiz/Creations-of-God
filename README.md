### Installation

```bash
npm install creations-of-god
```

or with [Yarn](https://yarnpkg.com)

```bash
yarn add creations-of-god
```

### Usage

CommonJS module:

```javascript
const { iamwebwiz } = require('creations-of-god');
```

or with ES6 module:

```javascript
import { iamwebwiz } from 'creations-of-god';
```
